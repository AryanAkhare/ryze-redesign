━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Purpose:
  Global navigation bar that appears on all pages with routing, dropdown menu,
  and theme toggle functionality.

Key Features:
  • Fixed positioning with scroll-based styling changes
  • Responsive design (hamburger menu on mobile, full nav on desktop)
  • "About Us" dropdown with hover activation
  • Theme toggle button with visual indicator
  • Smooth transitions on scroll (height, backdrop blur, scale)

Subtle Details:
  • Navbar height changes from 28 (7rem) to 20 (5rem) on scroll
  • Backdrop blur effect activates when scrolled
  • Logo scales slightly (0.97) when navbar is scrolled
  • Theme indicator dot shows current theme color
  • Dropdown menu has smooth opacity and translate animations
  • NavLink components provide active state styling

Theme Toggle:
  • Cycles through 5 themes: Default, Blue, Purple, Green, Dark
  • Updates CSS variables globally
  • Saves preference to localStorage
  • Loads saved theme on page load
  • Includes RGB variable for rgba() color usage

HEART ANIMATION (EASILY MISSED!)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Location: Testimonials section header, next to "Wall of Love" text

How It Works:
  • Pure CSS animation using keyframes
  • Heart shape created with ::before and ::after pseudo-elements
  • Base heart rotated -45 degrees
  • Two circles positioned to form heart lobes

Animation States:
  • Default: Slow heartbeat (1.6s cycle, scale 1.0 to 1.06)
  • Hover: Fast heartbeat (0.7s cycle, scale 1.0 to 1.15)

CSS Classes:
  • .heart-wrapper: Container for positioning
  • .heart: Base heart element with rotation
  • .heart::before, .heart::after: Circular lobes

Try It:
  Hover over the heart in "Wall of Love" header to see it beat faster!

Grid Layout:
  • Uses CSS Grid with named template areas
  • Responsive: stacks on mobile, grid on desktop
  • Cards positioned using grid-area property

Card Types:
  1. BigImageCard: Full-height image with gradient overlay
  2. ImageCard: Smaller image card
  3. QuoteCard: Black card with white text, inverts on hover
  4. StatCard: Metric display with hover shadow
  5. StatCard2: Logo + testimonial combo
  6. TestimonialCard: Social media-style card with Twitter blue hover

Hover Effects:
  • Quote card: Border appears, background inverts (black ↔ white)
  • Stat cards: Shadow intensifies with brand color
  • Testimonial cards: Twitter blue ring and shadow on hover
  • All cards: Smooth transitions (300ms ease-out)

                        4. THEME SYSTEM
================================================================================

CSS Variables (index.css):
  • --color-bg: Background color
  • --color-text-primary: Primary brand color (accent)
  • --color-text-secondary: Main text color
  • --color-accent: Accent color
  • --color-primary-rgb: RGB values for rgba() usage

Theme Toggle (Navbar.jsx):
  • Cycles through 5 themes
  • Updates all CSS variables instantly
  • Saves to localStorage
  • Loads on page load
  • Visual indicator dot shows current theme

Color Propagation:
  • All components use CSS variables
  • No hardcoded colors (except fallbacks)
  • Gradients use CSS variables
  • Opacity-based colors use RGB variable

================================================================================
                    5. ANIMATIONS & INTERACTIONS
================================================================================

Parallax Effects:
  • Background elements move slower than foreground
  • Different elements move at different rates
  • Creates depth and visual interest
  • Smooth spring physics via Framer Motion

Scroll Animations:
  • Elements fade in as they enter viewport
  • Staggered delays for sequential appearance
  • Y-axis transforms for slide-up effects
  • Opacity transitions for smooth reveals

Hover States:
  • Cards: Lift, scale, shadow increase
  • Buttons: Color change, scale, glow
  • Links: Color change, underline
  • Images: Grayscale removal, scale

Micro-interactions:
  • Button press-down on click
  • Input focus glow
  • Badge pulse animations
  • Progress bar shimmer
